## How to send them to me

1. Take all screenshots on your phone
2. Transfer to your computer (USB, Google Photos, AirDrop, etc.)
3. Drop them all into `attached_assets/screenshots/` in this project
4. Tell me they're uploaded — I'll process and slot each one into the right video

---

## Naming convention

All screenshots must be placed at:
```
attached_assets/screenshots/<filename>.png
```

The `@assets` alias in Vite already maps to `attached_assets/`, so no path changes needed in the code — just name the files exactly as listed above.

---

## Tips for good screenshots

- **Disable status bar** if possible (or enable immersive mode before capturing)
- **Use real data** — real app names, real times, real usage numbers look far more credible
- **Portrait orientation only** — all phone frames in the videos are portrait
- **Don't crop** — send the raw full-screen screenshot, I'll handle fitting it into the phone frame
- **Capture at a calm moment** — no notification banners, no battery warnings visible
